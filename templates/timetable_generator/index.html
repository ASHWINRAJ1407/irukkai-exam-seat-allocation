Smart Conflict Detection: The system now uses Subject Names (not codes) to prevent scheduling conflicts. For example, if both AIDS and AIML teach "Deep Learning", the system recognizes them as the same subject and limits them to 2 departments per day. Different codes for the same subject name? No problem—the system handles it!
Excel format — click to expand

Your Excel/CSV must have these columns (case-insensitive):

ColumnDescriptionRequired?
department_codeDepartment code (e.g. AIDS, AIML, CSE)✓ Yes
subject_codeSubject code (e.g. 19UADPEX05)✓ Yes
subject_nameSubject name (e.g. Deep Learning, ML Algorithms)Optional*

* If subject_name column is not provided, the system will use the Subject Code column to identify subjects.

Duplicate rows are kept: one row per (department, subject). Example:

department_code,subject_code,subject_name
AIDS,19UADPEX05,Deep Learning
AIDS,19UAMPEX06,Machine Learning
AIML,19UAMPEX05,Deep Learning
AIML,19UMBHS701,Data Science
CSE,19UCSPC702,Web Development

Note: When AIDS and AIML both teach code '19UADPEX05' with name "Deep Learning", both are counted as one subject for conflict resolution.
