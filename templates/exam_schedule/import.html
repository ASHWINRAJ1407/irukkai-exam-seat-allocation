To avoid import failures, ensure your file has exactly these column names (headers in first row). Column names are case-insensitive and spaces are allowed.

Column nameDescription
subject_codeMust match a subject code in Subjects (e.g. CS101)
department_codeMust match a department code in Departments (e.g. CSE)
exam_dateDate as YYYY-MM-DD or Excel date
start_timeTime as HH:MM or HH:MM:SS or Excel time
end_timeTime as HH:MM or HH:MM:SS or Excel time

Example (CSV):

subject_code,department_code,exam_date,start_time,end_time
CS101,CSE,2024-12-01,09:00,11:00
MA201,AIDS,2024-12-01,14:00,16:00
